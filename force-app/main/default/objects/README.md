# SentinelFlow Custom Objects

Create one folder per object in this directory using Salesforce DX source format.

Planned objects for SentinelFlow:

- `Incident__c`
- `Integration_Log__c`
- `SLA_Policy__c`
- `Environment__c`
- `Deployment_Record__c`
- `Audit_Trail__c`

Example structure:

```text
objects/
  Incident__c/
    Incident__c.object-meta.xml
    fields/
      Severity__c.field-meta.xml
```
