# Salesforce Metadata Source

This directory follows the standard Salesforce DX source layout.

Recommended metadata areas:

- `objects/`: custom objects, fields, validation rules, record types
- `classes/`: Apex classes and tests
- `triggers/`: Apex triggers
- `lwc/`: Lightning Web Components
- `permissionsets/`: permission sets
- `profiles/`: profiles when explicitly required
- `layouts/`: page layouts
- `tabs/`: custom tabs
- `flexipages/`: Lightning record and app pages

SEOMP-specific metadata should be added here as the Salesforce implementation grows.
